# Python Tic Tac Toe

A simple command-line Tic Tac Toe game written in Python while learning core programming fundamentals.

## Features
- Two-player gameplay
- Input validation
- Win detection
- Tie detection
- Replay option

## How to Run

1. Download the repository
2. Run the script:

python tic_tac_toe.py

## Concepts Practiced
- Functions
- Loops
- Conditional logic
- Program state management
